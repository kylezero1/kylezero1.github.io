Eight Puzzle Solver

The Eight Puzzle is a classic puzzle where numbers are laid out in a 3x3 table with the bottom right square blank. This program uses the A* Algorithm with either the Manhattan Distance Heuristic or the "Misplaced Tile Heuristic" and also Uniform Cost Search to solve any inputted eight puzzle. It solves the puzzle and also returns the number of nodes it takes.
